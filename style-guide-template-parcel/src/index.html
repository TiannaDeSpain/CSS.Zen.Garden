Style Guide 1.0

Purpose

The goal of this site is to collaborate as a team to work on learning more about CSS and SCSS. We have taken on the goal of creating a CSS Zen Garden site, using the content giving in the challenge. As a team we have decided to move forward with a ocean horizon theme. This will be implemented in color, font, style and depiction decisions. For the colors, we have decided that blues and oranges will help communicate the ocean theme. As for font, we will reflect the fun and bold theme. Depictions that we choose, such as SVG images or images created through CSS will be things like clouds, fish and/or other ocean/beach images. We will also have a touch of transparency that will help allude to the ocean theme, like water's transparency. Overall we think that this theme will challenge our learning while honing in our skills.

Typography

Headings Font - Fuzzy Bubbles

Body Font and other texts will be in Kodchasan. Both fonts can be found on Google Fonts, and have several options for weight of the text as well. We feel that these fonts will help communicate our theme of underwater horizons. They are easy on the eye, while also fitting in with the fun, bold theme.
